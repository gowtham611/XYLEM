## ðŸ’¡ Usage Examples

### In Widget Code:
```dart
final l10n = AppLocalizations.of(context);

// Simple text
Text(l10n?.xaiFeatureAnalysis ?? 'Feature Analysis')

// With dynamic values
Text('${l10n?.featureNitrogen ?? 'Nitrogen (N)'}: ${value.toFixed(1)}')

// With replacement
Text((l10n?.xaiWhyCrop.toString() ?? 'Why {crop}?').replaceAll('{crop}', cropName))

// In SnackBar
ScaffoldMessenger.of(context).showSnackBar(
  SnackBar(content: Text(l10n?.predictionSaved ?? 'Prediction saved to history'))
);
```

### In AlertDialog:
```dart
AlertDialog(
  title: Text(l10n?.clearAllPredictions ?? 'Clear All Predictions?'),
  content: Text(l10n?.clearAllConfirmation ?? 'This will permanently delete...'),
  actions: [
    TextButton(
      onPressed: () => Navigator.pop(context),
      child: Text(l10n?.cancel ?? 'Cancel'),
    ),
    TextButton(
      onPressed: () => _deleteAll(),
      child: Text(l10n?.deleteAll ?? 'Delete All'),
    ),
  ],
)
```

## ðŸ”„ Adding New Languages

To add more languages (e.g., Hindi):

1. Create `lib/l10n/app_hi.arb`
2. Copy all keys from `app_en.arb`
3. Translate values to Hindi
4. Run `flutter gen-l10n`
5. Update `l10n.yaml` if needed
